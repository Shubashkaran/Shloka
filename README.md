# Shloka
Face Mask Detection and Social Distancing Detection using Python and OpenCV.

Dependencies:
1. face-detection
2. flask
3. imutils
4. keras
5. matplotlib
6. opencv-python
7. pandas
8. pip-chill
9. scikit-learn
10. tensorflow

Installation:
1. Install the dependant packages using below command
     pip install -r requirements. txt
2. Run the main module
     python3 main.py
